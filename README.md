1. Project Title
🎵 Spotify Data Analytics Dashboard (Power BI)

2. Short Description / Purpose
This project is an interactive Spotify Data Analytics Dashboard built using Power BI to analyze music streaming data. It helps in understanding song popularity, artist performance, and overall music trends by converting raw data into meaningful visual insights.

3. Tech Stack

Power BI
Data Cleaning & Transformation
Data Modeling
Data Visualization

4. Data Source

Spotify dataset (public dataset / CSV file containing song, artist, album, and popularity details)

5. Features and Highlights

Interactive multi-page dashboard (Overview, Artists, Songs)
Key KPIs: Total Songs, Distinct Artists, Average Popularity, Duration
Analysis of top songs, top artists, and album performance
Trend analysis based on release year/month
Explicit vs Non-explicit song comparison
Drill-down and filtering for better user interaction
Clean and user-friendly dashboard design
